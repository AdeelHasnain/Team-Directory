Team Directory

A Team Directory app with MySQL, ColdFusion REST API, and React (Vite). Displays employees in a responsive UI with client-side search.

Tech Stack

MySQL – team_directory, Employees table

ColdFusion – REST API (GET /api/employees.cfm)

React + Vite – Fetch API, search, responsive layout

Features

List all employees (JSON)

Search by first/last name (case-insensitive)

Responsive: table (desktop), cards (mobile/tablet)

Secure queries with <cfqueryparam>
